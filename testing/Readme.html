Thank you for your interest in Numerical Catalyst.

Getting started

The installation script attempts to install the Numerical Catalyst add-in for you automatically, but if it becomes clear you need to do so activate this Excel add-in by selecting it from within the your version of Excel, as with any other add-in. This process varies slightly between Excel 2003 and earlier and Excel 2007 and latter and there is a section in the help file which outlines how to achieve this. (The help file can be accesses via the start menu options in the folder named at installation time.)

Macro security questions:

Since Numerical Catalyst is an add-in it doesn’t require the same macro security settings as normal workbook would.

However if you are having difficulties try altering the Trust centre settings for add-ins (Excel 2007 + ) to allow add-ins other than from signed sources and to deselect the block on all add-ins ( if it has been applied ). Note currently the add-in is not signed, however for a client that had this requirement this could be arranged.

I’m challenged for a licence whilst installing ( only on expired demo versions )

If you have a demo copy of Numerical Catalyst that has expired you are likely to be challenged for a licence during the installation, and each time Excel is started up. You need to use the licence information you will have been emailed as part of your purchase.

Purchase of a licence

A licence for Numerical Catalyst can be purchased online via our e-Commerce site at http://www.astonread.com/Home/sales , for orders requiring a different form of processing please contact sales@astonread.com .

Why does it say my licence is temporary when I’ve just purchased it ?

When you buy a licence you are sent a temporary 14 day licence. This is to cover the period of time that it takes to organise your machine licence. You need to open the options dialog and look at the about tab. A machine code will be displayed which must be mailed to support@astonread.com – its best to cut and paste the code to be sure. ( Note send from the same email address you used for the purchase ).

Demo version

The demo version is exactly as the sold version, except the ability to add a licence is not supported and there is a time limit. The time limit is currently about 1 year, to allow students in very low income areas to complete courses etc. Please consider buying the Continuing Professional Development version if you are earning a regular wage.

Continuing Professional Development version

Commercial version

This version is for all other uses and also allows for email technical support.

Registering and activating Numerical Catalyst

Numerical Catalyst deploys a form of DRM protection that requires a name and a code. These can be entered in the demo installation by opening options / about / licence. ( However it is perhaps best to do a fresh installation if you have just made a purchase with the latest installer as new options or examples may have been bundled since you ran the demo licenced version.
